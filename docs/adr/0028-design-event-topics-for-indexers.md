# ADR 0028: Design event topics for indexers

- Status: Accepted
- Deciders: arisu6804

## Context

The YieldVault smart contract needs a clear, documented approach to "design event topics for indexers" so the codebase stays consistent and auditable.

## Decision

We design event topics for indexers as the standard for this contract, in line with Soroban best practices.

## Consequences

Improves clarity, testability, and maintainability, and gives future contributors a recorded rationale to build on.
